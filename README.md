Nova Can Also be used as a unblocked games template just follow these steps

games.html **Main Template** You should name this index.html in your site

games-icons **Folder**
games **Folder**

games.json **Adds games into site**
Use this code
[
  "games/yourgame.html",
  "games/yourgame.html",
  "games/yourgame.html"
]
icons.json **adds icons into site**
{
  "games/crossyroad.html": "games-icons/yourgame.imagetype",
  "games/crossyroad.html": "games-icons/yourgame.imagetype",
  "games/crossyroad.html": "games-icons/yourgame.imagetype"
}
After following these steps your game template should be complete just rename things in the game.html and redo colors
